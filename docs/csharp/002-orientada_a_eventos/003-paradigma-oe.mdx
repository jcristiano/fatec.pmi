# Sobre o paradigma de programação orientada a eventos

Uma aplicação ou linguagem orientada a eventos é um paradigma de programação em que o sistema é projetado para responder a eventos e ações que ocorrem de forma assíncrona. Nesse contexto, eventos podem ser qualquer coisa, desde um clique do mouse até a chegada de dados em uma porta de rede. Um modelo de barramento é uma abordagem comum para implementar sistemas orientados a eventos, onde os eventos são publicados em um barramento (também conhecido como canal ou fila) e os consumidores interessados se inscrevem para receber notificações sobre esses eventos.
